# SyncPod-web-app
SyncPod for web

## Required Tools
* [yarn](https://yarnpkg.com/lang/ja/)
* [editorconfig plugin](http://editorconfig.org/#download)

## Setup
1. Execute the following command.
```sh
git@github.com:git@github.com:cyder/SyncPod-web-app.git
cd SyncPod-web-app
yarn
```

2. Start webpack-dev-server
```sh
yarn start
```
